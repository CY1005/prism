# ADR-008: 0.x 项目归个人，1.x 迁移到团队

## Status: Accepted

## Context

杨子老师等用户未来可能要在平台上创建自己的项目，需要团队/空间概念。但 MVP 阶段用户极少（1-5 人），团队模型会增加开发量。

## Decision

- 0.x 版本：项目归个人（project.owner_id = user_id）
- 1.x 版本：新增 team 表，项目归团队
- 迁移方案：自动为每个现有用户创建"个人团队"，项目挂过去

## Consequences

- **好处**：MVP 快速交付，不被团队模型拖慢
- **代价**：1.x 需要一次数据迁移
- **前提**：0.x 权限判断逻辑抽成独立模块，不散落各处，迁移时只改一处
