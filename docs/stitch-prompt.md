# Prism Stitch 原型生成提示词

> 创建时间：2026-04-03
> 用途：在 Google Stitch 中一次性生成 Prism 全部页面原型

---

我要设计一个完整的产品——Prism，产品全生命周期分析平台。请严格按照以下描述生成所有页面，不要添加任何我没有描述的功能或元素。

## 全局风格要求
- 风格：极简、高效、干净。参考 Linear / Notion 的视觉风格
- 配色：黑白灰为主，仅用一个强调色（蓝色）标识关键操作和选中状态。完善度用绿/黄/红三色，除此之外不要多余色彩
- 无装饰性元素，无渐变背景，无阴影过重的卡片
- 字体层级清晰（标题/正文/辅助文字三级），行间距舒适
- 大量留白，信息密度克制，一屏不要塞太多内容
- 边框用细线或无边框，用间距区分区块
- 所有页面保持统一的导航栏、配色、间距、圆角风格
- 适合桌面端（1440px）
- 所有文字使用中文

## 全局导航栏（所有页面顶部固定）
- 左侧：Prism logo + 产品名
- 中间：全局搜索框（占位文字"搜索功能、模块、竞品..."）
- 右侧：通知图标 + 用户头像 + 用户名"陈玥"

- 居中登录卡片，背景简洁

- 每个卡片内容：

  - 一句话描述

## 页面间跳转关系
- 登录页 → 登录成功 → 项目列表页
- 项目列表页 → 点击项目卡片 → 项目全景图
- 项目列表页 → 点击新建项目 → 新建项目弹窗
- 项目全景图 → 点击功能节点 → 功能模块档案页
- 功能模块档案页 → 左侧树切换 → 更新右侧内容
- 全局搜索 → 回车 → 搜索结果页
- 搜索结果页 → 点击某条 → 功能模块档案页
- 导航栏用户头像 → 下拉菜单 → 项目设置 / 运维后台 / 退出登录
