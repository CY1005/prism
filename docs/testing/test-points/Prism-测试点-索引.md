## 各版本核心关注点

### v0.1 MVP — 能录、能看、能看见结构
- **重点**：CRUD正确性、权限隔离（4角色）、树形导航、维度卡片动态渲染、乐观锁
- **高风险**：节点删除策略（PRD未明确）、注册流程（设计稿未实现）

### v0.2 — 能分析、能对比、能追踪
- **重点**：AI服务稳定性（降级策略）、分析结果写入维度、测试点生成质量
- **高风险**：AI服务不可用时的降级是否真正不影响CRUD、分析结果写入不破坏已有数据
- **回归**：档案页、搜索、完善度计算

### v0.3 — AI增强
- **重点**：智能导入7种AI能力（拆分/归类/提取/补全/关联/去重/差异标注）、搜索底层切换（ILIKE→pgvector）
- **高风险**：搜索底层切换影响全局、撤销导入的数据一致性、embedding同步机制
- **回归**：搜索全面回归、CSV导入共存

### v1.x — 团队化
- **重点**：项目归属模型变更（个人→团队）、双层权限模型、数据库迁移
- **高风险**：迁移策略（自动建个人团队+项目挂载）的数据完整性、权限叠加规则
- **回归**：全面回归（底层数据模型变更影响所有功能）
