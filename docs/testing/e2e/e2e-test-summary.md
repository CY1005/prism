## 阻塞性问题（必须修复才能发布）

### Critical（2 个）

1. **BUG-075: /search/unified 无认证保护** — 任何人可无 Token 访问搜索接口，泄露项目数据。修复成本低（添加 Depends(require_user)）。
2. **BUG-076: Viewer 角色可创建项目** — 权限模型被绕过，低权限用户可以创建和操控项目。修复需在 POST /api/projects/ 添加角色检查。

### High（影响核心链路，2 个）

3. **BUG-069: 无节点 CRUD 端点** — P0 冒烟测试失败。用户无法通过 API 创建/修改/删除功能项节点，F3 功能模块树的核心写入能力缺失。
4. **BUG-070: Issues 表列名不一致且无 CRUD 端点** — F7 问题沉淀功能完全不可用。hybrid_search 中的 SQL 引用错误列名，查询 issue 时会报错。
