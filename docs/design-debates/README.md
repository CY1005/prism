# Design Debates（设计辩论记录）

这个目录记录 Prism 重要设计决策的**完整辩论过程**——不只是最终结论，还包括：

- 当时想解决什么问题
- 有哪些候选方案
- 谁提出了什么质疑
- 为什么选择 A 不选 B
- 承认了什么错误、踩了什么陷阱

## 为什么要保留过程

最终规则文档（`docs/architecture/` 下）回答"**是什么**"；
辩论记录回答"**为什么是这样，不是别的样**"。

3 个月后有人质疑某条规则——翻这里能看到当时的场景和担忧，而不是让他重走一遍弯路。
