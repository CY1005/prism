# Prism 业务逻辑与流程清单（基于v0设计稿）

> 来源：v0 原型设计稿 13个页面逐页提取
> 更新时间：2026-04-10
> 用途：需求文档补充、测试点设计输入

---

## 页面导航总图

```
登录页 (/login)
  ↓
项目列表 (/projects) ← 全局返回点
  ├→ 新建项目 (/projects/new)
  ├→ 搜索 (/search)
  ├→ 运维后台 (/admin)
  └→ 项目详情 (/projects/[id])
      ├─ [Tab] 全景图（当前页）
      ├─ [Tab] 产品线概览 (/projects/[id]/product-lines/[plId])
      │   └→ 模块概览 (/projects/[id]/modules/[moduleId])
      │       └→ 功能档案页 (产品分析 / 系统架构)
      ├─ [Tab] 需求工作台 (/projects/[id]/analysis)
      ├─ [Tab] 竞品对比 (/projects/[id]/comparison)
      ├─ [Tab] 设置 (/projects/[id]/settings)
      └→ 导入 (/projects/[id]/import)
```

## 1. 登录/注册

### 正向流程
| # | 操作 | 预期结果 |
|---|------|----------|
| 1.1 | 输入有效邮箱+正确密码 → 点击登录 | 跳转到 /projects |
| 1.2 | 点击"没有账号？注册" | 跳转到注册流程（设计稿未实现） |

### 负向流程
| # | 操作 | 预期结果 |
|---|------|----------|
| 1.3 | 邮箱为空，点击登录 | 提示"请输入邮箱" |
| 1.4 | 密码为空，点击登录 | 提示"请输入密码" |

| 1.6 | 邮箱正确+密码错误 | 提示"邮箱或密码错误" |

| 1.10 | 登录后Session过期 | 下次请求重定向到 /login |

## 3. 新建项目

### 正向流程
| # | 操作 | 预期结果 |
|---|------|----------|
| 3.1 | 输入项目名称 + 描述 + 选择模板 → 点击创建 | 创建项目，写入dimension_config和hierarchy_labels，跳转到项目全景图 |

## 8. 需求工作台（AI分析）

### 正向流程
| # | 操作 | 预期结果 |
|---|------|----------|
| 8.1 | 左侧输入区 | 需求描述文本框 + 3个工具按钮（文字/文档/图片） |
| 8.2 | 输入需求文字 | textarea支持拖拽上传提示 |
| 8.3 | 上传文档 | 点击FileText按钮 → 选择文件(.pdf/.doc/.docx/.txt) → 显示文件Badge |
| 8.4 | 上传图片 | 点击ImagePlus按钮 → 选择图片(.png/.jpg/.jpeg) → 显示图片Badge |
| 8.5 | 拖拽上传 | 拖拽文件到输入区 → 自动判断类型(文档/图片) → 添加到对应列表 |
| 8.6 | 文件上传中 | 显示Loader2旋转动画，800-1300ms后完成 |
| 8.7 | 删除已上传文件 | 点击文件Badge上的X → 从列表移除 |
| 8.8 | 点击"AI分析" | 1.5s模拟延迟 → 右侧显示分析结果 |
| 8.9 | 分析结果展示 | 4张卡片：影响范围（直接+间接）/ 完整性评估 / 合理性评估 / 用户视角评估 |
| 8.10 | 影响范围卡片 | 直接影响：Badge+路径+原因；间接影响：同上 |
| 8.11 | 完整性评估 | ✓/✗ + 评估项文本 |
| 8.12 | 合理性评估 | ✓/✗/⚠ + 评估项文本 |
| 8.13 | 用户视角评估 | 按角色分组，每个角色下列出concerns |
| 8.14 | 点击"保存分析结果" | 结果写入对应功能项的"需求分析"维度 |
| 8.15 | 点击"生成测试点" | AI基于分析结果生成测试点列表 |

### 负向流程
| # | 操作 | 预期结果 |
|---|------|----------|
| 8.16 | 无输入内容，点击AI分析 | 按钮disabled，不可点击 |
| 8.17 | 正在分析中 | 按钮显示loading状态，不可重复点击 |
| 8.18 | 文档超过3个 | 文档上传按钮disabled |
| 8.19 | 图片超过5个 | 图片上传按钮disabled |
| 8.20 | 上传不支持的文件格式 | 提示"不支持的文件格式" |
| 8.21 | AI分析失败/超时 | 显示错误提示"分析服务暂时不可用" |
| 8.22 | 分析结果为空 | 显示"未找到相关影响" |
| 8.23 | 未分析就点保存 | 按钮disabled或不显示 |
