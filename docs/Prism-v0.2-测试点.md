## F12：功能对比矩阵

### 功能测试

#### 表单验证
- 竞品对比-功能选择-验证下拉菜单可选择功能项（创建推理服务/自动扩缩容等） [P0]
- 竞品对比-添加竞品-验证点击"+ 添加竞品"后可选择或输入竞品名称 [P0]
- 竞品对比-竞品Badge-验证已选竞品显示为Badge组（本产品+竞品1+竞品2），竞品可删除(X) [P0]

#### 对比生成与展示
- 竞品对比-生成对比-验证点击"生成对比"后AI生成对比表格 [P0]
- 竞品对比-对比表格-验证列结构正确（对比维度/本产品/竞品A/竞品B） [P0]
- 竞品对比-差异高亮-验证差异项颜色标注正确（绿=优势，红=劣势） [P0]
- 竞品对比-对比结论-验证AI生成优势/劣势列表（✓=优势绿色，⚠=劣势橙色） [P1]
- 竞品对比-结果导出-验证对比结果可导出（格式待确认） [P1]
- 竞品对比-选择2+竞品/产品线/版本-验证均可作为对比维度 [P1]

#### 负向流程
- 竞品对比-未选择功能-验证"生成对比"按钮disabled [P0]
- 竞品对比-竞品数量<2-验证提示"至少选择2个对比对象" [P1]
- 竞品对比-删除所有竞品-验证清空Badge回到初始状态 [P1]
- 竞品对比-对比生成失败-验证显示错误提示 [P1]

## F13：AI需求分析

### 功能测试

#### 需求输入区
- 需求工作台-左侧输入区-验证显示需求描述文本框+3个工具按钮（文字/文档/图片） [P0]
- 需求工作台-文字输入-验证textarea支持拖拽上传提示 [P1]
- 需求工作台-上传文档-验证点击FileText按钮可选择文件(.pdf/.doc/.docx/.txt)并显示文件Badge [P0]
- 需求工作台-上传图片-验证点击ImagePlus按钮可选择图片(.png/.jpg/.jpeg)并显示图片Badge [P0]
- 需求工作台-拖拽上传-验证拖拽文件到输入区自动判断类型并添加到对应列表 [P1]
- 需求工作台-文件上传中-验证显示Loader2旋转动画 [P1]
- 需求工作台-删除已上传文件-验证点击Badge上X后从列表移除 [P1]

#### AI分析流程
- 需求工作台-点击AI分析-验证右侧显示分析结果（4张卡片） [P0]
- 需求工作台-影响范围卡片-验证显示直接影响（Badge+路径+原因）和间接影响 [P0]
- 需求工作台-完整性评估-验证显示✓/✗+评估项文本 [P0]
- 需求工作台-合理性评估-验证显示✓/✗/⚠+评估项文本 [P0]
- 需求工作台-用户视角评估-验证按角色分组展示concerns [P0]
- 需求工作台-影响范围高亮-验证分析结果中的影响模块在关系图上高亮显示 [P1]

#### 分析结果操作
- 需求工作台-保存分析结果-验证结果写入对应功能项的"需求分析"维度 [P0]
- 需求工作台-生成测试点-验证AI基于分析结果生成测试点列表 [P0]
- 需求工作台-测试点review-验证生成的测试点可逐条review [P1]
- 需求工作台-测试点一键录入-验证review后可一键录入到功能项的"测试分析"维度 [P0]

#### 负向流程
- 需求工作台-无输入点击AI分析-验证按钮disabled不可点击 [P0]
- 需求工作台-分析进行中-验证按钮显示loading状态不可重复点击 [P0]
- 需求工作台-文档超过3个-验证文档上传按钮disabled [P1]
- 需求工作台-图片超过5个-验证图片上传按钮disabled [P1]
- 需求工作台-上传不支持格式-验证提示"不支持的文件格式" [P1]
- 需求工作台-AI分析失败/超时-验证显示错误提示"分析服务暂时不可用" [P0]
- 需求工作台-分析结果为空-验证显示"未找到相关影响" [P1]
- 需求工作台-未分析点保存-验证按钮disabled或不显示 [P1]

### 接口测试

#### FastAPI分析接口
- POST /analyze-正常请求-验证返回结构化分析结果（影响范围+完整性+合理性+用户视角） [P0]
- POST /analyze-description+context-验证上下文信息参与分析（关联已有知识库内容） [P1]
- POST /test-points-正常请求-验证返回结构化测试点列表（含优先级+分类） [P0]
- POST /compare-正常请求-验证返回对比表格数据 [P0]
- POST /analyze-Claude API限流-验证返回503且前端降级提示 [P1]
- POST /analyze-请求超时(30s)-验证超时处理正确 [P1]
